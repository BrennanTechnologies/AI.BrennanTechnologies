# M365 Governance Workflow Guidance

- Use least privilege role assignments.
- Sensitive user lifecycle actions require explicit admin approval.
- Incident response actions should be logged with actor, timestamp, and outcome.
- Teams notifications should be posted to approved governance channels.
- Remediation summaries must include evidence and owner.
